## Cover Page Design

The cover page is critical. Follow this specification exactly:

### Background
- Full-page dark purple gradient: `#1a0f2e` to near-black (`#0a0a0f`)
- No decorative elements (no grids, no sun, no textures) - keep it clean

### Title Text
- English title: "Retro Tech Art Style" in very large font
- Gradient effect: pink (`#ff2aa3`) -> cyan (`#00f5ff`) -> purple (`#7b2cff`) left-to-right
- Centered horizontally
- Positioned in upper-middle area (roughly 30-40% from top)
- A thin purple horizontal line directly beneath the title

### Subtitle Text
- Chinese subtitle: "复古科技艺术风格设计理念完整指南"
- Color: pink/magenta (`#ff2aa3`)
- Centered, directly below the title line
- Medium-large font size

### Date Line
- Text: "2026年4月" (adjust date as needed)
- Color: cyan (`#00f5ff`)
- Centered, below subtitle with generous spacing
- Medium font size

### No Extra Title Page
- After the cover page, go directly to Table of Contents
- Do NOT add a separate title page after the cover

## Page Layout

### Header (All Content Pages)
- Content: "Retro Tech Art Style 设计理念指南"
- Position: Centered at top of every content page
- Style: Small, regular weight, subtle dark gray or black
- NOT bold, NOT large - should be understated

### Footer (All Content Pages)
- Content: Page number only
- Position: Centered at bottom of page
- Style: Small, regular weight
- No copyright text, no additional decoration

### Content Page Background
- White background for all content pages (pages 3 onwards)
- Generous margins on all sides

### Table of Contents Page
- Single page only
- "目录" heading with purple gradient treatment and left purple vertical bar accent
- List chapter titles with right-aligned page numbers
- Format: "第一章：风格概述 .................. 3"
- Only chapter-level entries (no sub-sections in TOC)
- Clean, spaced layout - do not crowd entries

### Chapter Heading (H1)
- Format: "第X章：[Title]" in Chinese
- Large purple text with gradient effect (purple `#7b2cff` to pink `#ff2aa3`)
- Full-width purple gradient underline directly below heading
- Page break before each chapter heading
- Examples from reference:
  - "第一章：风格概述"
  - "第二章：历史发展"
  - "第三章：核心设计哲学"
  - "第四章：视觉元素详解"
  - "第五章：子风格对比"
  - "第六章：现代应用案例"
  - "第七章：创作工具与资源"
  - "第八章：经典配色方案"
  - "第九章：创作技巧"
  - "第十章：总结与参考"

### Section Heading (H2)
- Format: "X.Y [Title]"
- Pink/magenta (`#ff2aa3`) text
- Left vertical border accent (4px pink/magenta bar on left side)
- Medium-large font, bold
- No page break before

### Body Text
- Black (`#1a1a1a`) on white background
- Regular weight, readable size
- Key terms in **bold purple** (`#7b2cff` or `#6c5ce7`) for emphasis

### Callout/Definition Boxes
- Light tinted background (very subtle gray `#f8f9fa` or accent at 5% opacity)
- Left border accent (4px, purple `#7b2cff` or pink `#ff2aa3`)
- Used for: Key definitions, core concepts, concluding remarks
- Example: "核心定义：" label in purple, followed by definition text

### Tables
- Header row: Dark slate gray (`#2d3436`) background with white text
- Body rows: White background with light gray (`#f8f9fa`) alternating
- Clean horizontal separators between rows
- No vertical borders between columns (or very subtle)
- Cell padding for comfortable reading

### Bullet Lists
- Black round bullet markers
- First-level items only
- Key term in **bold purple**, followed by description in regular black
- Example: "**粗体无衬线：** Commando、Hauser - 标题和Logo"

### Numbered Lists
- Arabic numerals with period
- Used for step-by-step procedures
- Number in regular text, content in regular text

### Color Palette Display (for Chapter 8 style content)
- Color swatches displayed as horizontal gradient bars
- Each bar shows 5 color segments with hex values overlaid
- Colors transition smoothly between segments
- Hex values in small white or dark text centered on each segment

### Usage Rules
- Dark backgrounds (#1a0f2e or #0b0b0f) for cover page only
- White backgrounds for all content pages
- Neon accents for highlights, borders, glowing elements
- Yellow/orange for emphasis points and call-to-action elements
- Purple gradient for H1 headings and underlines
- Pink/magenta for H2 headings and left border accents

- Off-white for readable body text on dark backgrounds only

## Typography

### Reference Font
- **Primary**: LiberationSerif (serif family from PDF reference)
- **Style**: Clean, readable serif with moderate weight contrast

### Font Categories for Retro Tech Art

| Category | Example Fonts | Usage |
|----------|--------------|-------|
| Bold sans-serif | Commando, Hauser, Chunky Retro | Titles, logos, headlines |
| Brush/script | Streamster, Road Rage | Subtitles, decorative text |
| Pixel font | VCR OSD Mono | UI elements, retro game text |
| Neon effect | Neon 80s | Decorative glowing text |
| Modern retro | Atlantico | Body text, general content |

### Typography Rules
- Cover title: Very large, rainbow gradient (pink->cyan->purple)
- Cover subtitle: Medium-large, pink/magenta
- Cover date: Medium, cyan
- H1 headings: Large purple gradient with bottom border line
- H2 headings: Medium, pink/magenta with left border accent
- Body: Regular weight, black on white
- Bold key terms: Purple accent color inline
- Decorative text: Script or pixel fonts for thematic elements
- Ensure CJK support: Use Noto Sans CJK SC or equivalent for Chinese content

## Visual Elements

### Graphic Elements

| Element | Description | Application |
|---------|-------------|-------------|
| Laser grid | Glowing neon grid lines to vanishing point | Space/depth creation |
| Retro sun | Large sunset with horizontal stripe gradient | Background focal point |
| Wireframe | Low-polygon 3D rendered style | Tech/sci-fi elements |
| Pixel art | 8-bit / 16-bit game style patterns | Retro gaming themes |
| Palm tree silhouette | Tropical Miami silhouette | Decorative, atmospheric |
| Chrome surface | High-reflection metallic with gradient | Text treatments, logos |
| CRT scanlines | Horizontal line overlay | Vintage screen effect |
| VHS glitch | RGB channel separation, tearing | Distortion aesthetic |
| Film grain | Random noise texture | Analog warmth |
| Halftone dots | Dot pattern overlay | Comic/print effect |

### Element Application Priority
1. Establish background (dark gradient)
2. Add primary graphic element (grid, sun, scene)
3. Apply accent lighting (neon glows, highlights)
4. Overlay texture (scanlines, grain, halftone)
5. Add text/decorative elements

## Texture Effects

| Effect | Technique | Intensity |
|--------|-----------|-----------|

| RGB phosphor | Offset RGB channels slightly | Very subtle |

## Sub-Styles

### Synthwave
- **Era**: 2000s-present
- **Colors**: Pink, cyan, purple gradient
- **Elements**: Neon grids, sports cars, sunset
- **Mood**: Nostalgic energy
- **Use for**: Music artwork, game covers, posters

### Vaporwave
- **Era**: 2000s-present
- **Colors**: Pink-purple, teal-green
- **Elements**: Classical statues, glitch, consumer culture symbols
- **Mood**: Ironic melancholy
- **Use for**: Art prints, album covers, aesthetic wallpapers

### Cyberpunk
- **Era**: 1980s-present
- **Colors**: Neon on black, high contrast
- **Elements**: Rainy streets, skyscrapers, neon signs, augmented humans
- **Mood**: Dystopian, noir
- **Use for**: Character art, comic panels, scene illustrations

### Y2K Futurism
- **Era**: 1997-2004
- **Colors**: Ice blue, silver-white
- **Elements**: Transparent materials, metallic surfaces, organic shapes
- **Mood**: Tech optimism
- **Use for**: UI mockups, product visualization

### Frutiger Aero
- **Era**: 2004-2013
- **Colors**: Blue-green, white
- **Elements**: Glossy surfaces, nature motifs, butterflies
- **Mood**: Fresh, friendly
- **Use for**: Brand design, icon sets

### Atom Punk
- **Era**: 1940s-1960s
- **Colors**: Orange-yellow, silver
- **Elements**: Atomic symbols, rockets, space optimism
- **Mood**: Optimistic future
- **Use for**: Posters, illustrations

### Retro Comic
- **Era**: 1950s-1980s
- **Colors**: Bold primaries, halftone shading
- **Elements**: Action lines, speech bubbles, Ben-Day dots
- **Mood**: Dynamic, expressive
- **Use for**: Comic panels, storyboards, pop art
