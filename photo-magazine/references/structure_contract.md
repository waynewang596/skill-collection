# Structure Contract — Corporate Impact Report

Extracted from Microsoft Environmental Sustainability Report 2024.

## Document Architecture

### Overall Structure (88 pages)
```
Cover Page
Table of Contents ("In this report") + Key Statistics Highlights
Section 1: Overview
  - Foreword (leadership letter)
  - 2023 Highlights (data dashboard)
  - How we work (framework explanation)
Section 2: Microsoft Sustainability (operational footprint)
  - Carbon
  - Water
  - Waste
  - Ecosystems
Section 3: Customer Sustainability
  - Microsoft Cloud for Sustainability
  - Devices
  - Green software
  - Arts and culture
  - Planetary Computer
Section 4: Global Sustainability
  - Investing in innovation
  - Accelerate sustainability solutions with AI
  - Policy and advocacy
  - Employee engagement
  - Scaling impact through Green Skilling
Section 5: Appendix
  - Appendix A: How we report
  - Appendix B: Methodologies
Back Cover
```

### Section Hierarchy
1. **Major Sections** (5): Overview | Microsoft sustainability | Customer sustainability | Global sustainability | Appendix
2. **Themes** (under each major section): 4-6 thematic pillars
3. **Topics** (under each theme): 3-5 specific focus areas
4. **Sub-topics**: Individual initiatives, programs, or data points

## Page Type Templates

### Type A: Cover Page
- Full landscape, split 40/60 or 50/50
- Left: Logo + Large title (H1, light weight) + Report name + Date
- Right: Full-height environmental photograph

### Type B: Table of Contents
- Multi-column grid (5 columns, one per major section)
- Each column: Section name + list of topics with page numbers
- Bottom row: Large statistic highlight cards (4-5 cards across)
- Right column: Reporting disclosure text

### Type C: Section Opener / Divider
- Left panel (40%): Section label + H1 title with green underline + intro paragraph + mini TOC
- Right panel (60%): Full-height photograph with photographer credit
- Top navigation shows active section

### Type D: Content Page — Multi-Column
- Top: Section breadcrumb (e.g., "Carbon | Water | Waste | Ecosystems") with active bold
- Top-right: Page number
- Body: 3-4 column layout with vertical dividers
- Left column: H2 title + summary paragraph
- Center/right columns: Body text with H3/H4 subheadings, bullet lists
- Inline images with captions

### Type E: Data Dashboard Page
- 4-column grid of data cards
- Each card: Category title + Big metric number + unit + analogy/explanation + illustration
- Bottom: Bullet points with key achievements

### Type F: Target/Progress Page
- Two-column layout: "Our targets" | "Our progress"
- Each row: Bold target title + description → Green checkmark + Progress description
- Bottom: Large data highlight card spanning full width

### Type G: Story/Profile Page
- Large photograph with overlay text or adjacent text
- Employee/partner profile with photo, name, title
- Quote or testimonial in large serif text

### Type H: Appendix Page
- Dense multi-column text
- Subsections with H3 headings
- Legal/disclosure text
- Links to external resources

### Type I: Back Cover
- Full-bleed landscape photograph
- Left or right panel: Call to action text + links
- Logo + report name at bottom

## Heading Conventions
- **H1**: Appears only on section opener pages, large thin-weight text with green underline
- **H2**: Page-level titles, sentence case
- **H3**: Subsection titles within pages, bold
- **H4**: Paragraph-level headings, bold, smaller
- **Section breadcrumb**: Below nav, shows theme tabs with current bold

## Cross-Reference System
- **Page numbers**: Right-aligned with hamburger menu icon ("☰ 04")
- **TOC references**: Topic name + page number, dot leaders
- **Internal links**: Underlined text linking to other report sections
- **External links**: Underlined blue text with arrow icon

## Exhibit & Data Numbering
- Tables: "Waste Table 2 — Improving reuse and recycling..."
- Charts: Labeled with legend (A, B icons + labels)
- Data cards: Not numbered, identified by category + metric

## Header/Footer Rules
- **Header**: Navigation tabs (consistent across all pages)
- **Breadcrumb**: Below nav, shows current theme path
- **Footer**: None (page number is in header area)

## Narrative Flow Pattern
1. **Open with a question** — Each major section begins with an aspirational question ("How can we advance sustainability?")
2. **State the challenge** — Describe the global problem context
3. **Present commitments** — Company's goals and targets
4. **Report progress** — Data, metrics, achievements
5. **Acknowledge gaps** — Transparent about challenges and areas not on track
6. **Share learnings** — What worked, what didn't, lessons learned
7. **Look forward** — Future plans and "what's next"

## Content Density Guidelines
- **Section openers**: Low density — large text, big photos, minimal content
- **Content pages**: Medium-high density — multi-column text, multiple subsections
- **Data pages**: Medium density — visual statistics with supporting text
- **Appendix**: High density — compact text, legal language

## Front Matter / Back Matter
- **Front matter**: Cover, TOC (no separate title page, copyright page)
- **Back matter**: Appendix sections, back cover (no index, no glossary)

## Photography & Visual Asset Strategy
- **Hero images**: Full-page environmental/nature photography on section openers
- **Inline images**: 2-3 per content page, showing real work/people/technology
- **Employee photography**: Circular portraits for leadership/foreword
- **Illustrations**: Isometric/flat illustrations for data explanation pages
- **Image credits**: Every photograph credited to photographer with name, title, location
